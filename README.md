# SongGuesser

Song guessing bot.

## Installing

1. Make sure you have node.js installed.
2. Install the packages with `npm i`
3. Create a config.json file with the following contents:
   `
    {
      "token": "TOKEN",
      "client_id": "CLIENT_ID",
      "guild_id": "GUILD_ID (optional)"
    }
    `
   `
4. Update the commands with `node update.js`
5. Run it with `node .`
